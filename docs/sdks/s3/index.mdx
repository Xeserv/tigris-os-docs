# AWS S3 SDKs

Tigris is a S3-compatible object storage service. This means that you can
continue using the standard AWS S3 SDKs and libraries as usual. The
S3-compatible API allows you to access the wide range of available S3 tools,
libraries, and extensions.

Below you can find examples of how to use Tigris with the most popular S3 SDKs
and CLIs.
